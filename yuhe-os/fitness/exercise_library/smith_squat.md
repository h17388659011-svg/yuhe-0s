# 史密斯深蹲

目标肌群：股四头肌、臀部、核心。

## 当前个人数据

- 当前阶段：空杆学习。
- 最近完成：空杆3组8次。
- 第三组开始舒服，前两组仍在找感觉。

## 个人关键经验

早期问题：蹲的过程中杠顶脖子。

解决：握距比肩宽后，上背更稳定，杠不再顶脖子。

## 动作口诀

胸朝前，脚踩地。

## 动作要点

1. 杠放在上斜方肌，不压脖子。
2. 握距比肩略宽，找到不顶脖子的稳定位置。
3. 胸自然挺起，不能弓背。
4. 脚比杠略向前一点。
5. 下蹲时屁股向后坐，起身时脚踩地。

## 常见错误

- 杠压到脖子。
- 握距太窄导致上背撑不起来。
- 弓背。
- 一开始就加重量。
